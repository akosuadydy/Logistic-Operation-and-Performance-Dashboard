# Logistics Operation and Performance Dashboard

## Project Overview

This project is an interactive Power BI dashboard built to monitor logistics operations and business performance. It provides a clear view of revenue, operating cost, profitability, delivery distance, and customer activity, helping evaluate operational efficiency and support strategic decision-making.

Country: United States of America

## Dashboard Objectives

* Track overall logistics performance and profitability
* Monitor revenue and operating cost trends
* Analyze delivery operations across customer and booking categories
* Evaluate operational efficiency using transport and profit metrics

## Key Metrics

* Gross Revenue:$299M
* Total Operating Cost:175M
* Total Profit:124M
* Total Distance Covered: 122M
* Revenue Yield per Mile:$2.44
* Net Operating Profit Margin:42%

## Dashboard Analysis Sections

* Revenue by Customer Type – shows revenue contribution by customer category
* Revenue by Month – tracks monthly performance trends
* Revenue Generating City– highlights top-performing cities
* Revenue by Booking Type – compares booking channel performance
* Delivery by Customer Type– analyzes delivery patterns by customer segment
